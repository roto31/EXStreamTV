Streaming Settings

Configure stream buffer sizes and optional rate throttling for channel playback.

Stream Buffer
Size of the in-memory buffer for stream reads. Default: 2 MB (2097152). Range: 64 KB–32 MB. Larger values can smooth playback on slow networks.
Chunk size for each read from the source. Default: 64 KB (65536). Range: 4 KB–1 MB.
Stream Throttler
Rate-limit MPEG-TS delivery to the target bitrate to prevent client buffer overruns and improve stability.
Target bitrate for throttling. Default: 4 Mbps. Stored as bits per second internally.
Realtime: match playback rate. Burst: allow bursts up to buffer size. Adaptive: adjust from client feedback. Disabled: no throttling.
info Information

Stream buffer settings affect how much data is buffered before sending to clients. Stream throttler limits the send rate to avoid overwhelming players (e.g. Plex) and can reduce transcoding load. Changes take effect for new streams; existing streams keep their current settings until reconnected.
