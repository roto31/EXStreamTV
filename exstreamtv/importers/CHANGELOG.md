# Importers Component Changelog

All notable changes to the Importers component will be documented in this file.

## [2.5.0] - 2026-01-17
### Changed
- No changes to importers module in this release

## [1.0.6] - 2026-01-14
### Added
- M3U importer (`m3u_importer.py`)
- M3U discovery (`m3u_discovery.py`)
- M3U metadata extraction (`m3u_metadata.py`)
- Channel importer (`channel_importer.py`, `channel_importer_v2.py`)
- ErsatzTV database importer (`ersatztv_importer.py`)
- StreamTV database importer (`streamtv_importer.py`)
- Schema mapper for database compatibility (`schema_mapper.py`)
- Enum maps for database migrations (`enum_maps.py`)
- Validators for import data (`validators.py`)
