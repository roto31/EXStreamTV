# Documentation Screenshots

This directory contains screenshots for the EXStreamTV documentation guides.

## Screenshot Requirements

All screenshots should:

- Be captured at **1920x1080** or higher resolution
- Use a **clean browser window** (no bookmarks bar or extensions visible)
- Use the **default light theme** for consistency (dark theme variants optional)
- Have key elements **highlighted with red boxes** or arrows where helpful
- Include **descriptive alt text** in the documentation
- Be saved as **PNG format** with descriptive filenames

## Annotation Guidelines

When adding annotations to screenshots:

1. **Red boxes** - Use `#FF3B30` (Apple red) for highlight boxes
2. **Arrows** - Simple arrows pointing to key elements
3. **Numbers** - Use numbered callouts (1, 2, 3) for step sequences
4. **Text labels** - Brief text annotations in a clean sans-serif font

## Tools for Screenshot Capture

Recommended tools:

- **macOS**: Built-in Screenshot (Cmd+Shift+4) + Preview for annotations
- **Cross-platform**: Snagit, Greenshot, or ShareX
- **Browser extension**: Awesome Screenshot, Nimbus Screenshot

## Naming Convention

Use lowercase with hyphens:
- `{section}-{feature}-{description}.png`
- Examples: `nav-sidebar-full.png`, `channel-ai-conversation.png`

## Updating Screenshots

When the UI changes:

1. Capture new screenshot at same resolution
2. Apply same annotation style
3. Replace old file with same filename
4. Verify all documentation references still work
