# EXStreamTV MCP Server – Instructions for AI

Use this server when working on the EXStreamTV project. It exposes:

- **get_project_info** – Project summary, version, features, doc paths.
- **search_documentation** – Search docs/ and README by text query.
- **get_doc** – Full content of a doc by path or short name (e.g. INSTALLATION, QUICK_START, AI_SETUP).
- **list_docs** – List documentation files with path and title.
- **get_config_schema** – config.example.yaml and section overview.
- **get_api_overview** – API modules and main route categories.

Resources: `exstreamtv://docs/README`, `exstreamtv://config/example`.

Prefer these tools when answering questions about installation, configuration, API structure, or where to find documentation in the repo.
